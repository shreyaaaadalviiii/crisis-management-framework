# Delivery Management Dashboard & RAID Log Tracker

## Overview

The Delivery Management Dashboard is a comprehensive project tracking solution designed to provide real-time visibility into Agile project execution. It combines sprint tracking, RAID log management, milestone monitoring, team capacity planning, and delivery health reporting into a single dashboard.

Developed using Microsoft Excel, the dashboard demonstrates practical delivery management techniques through a simulated 10-month Mars Rover Science Team project involving 11 cross-functional team members.

---

## Objectives

* Monitor sprint progress and velocity
* Track project milestones and deliverables
* Identify and manage project risks
* Monitor blockers and dependencies
* Analyze team capacity and workload
* Improve delivery transparency

---

## Features

* Interactive Delivery Dashboard
* Sprint Velocity Tracking
* RAID Log (Risks, Assumptions, Issues & Dependencies)
